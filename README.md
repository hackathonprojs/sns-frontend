# Sei Naming Service Frontend


## To run

`npm install`

`npm run dev`

navigate to `http://localhost:5173/` to view your application


## How was it created


`npm create vite sns-frontend --template react-ts`



Install dependencies

`cd sns-frontend`

`npm install && npm install @sei-js/core @sei-js/react`
